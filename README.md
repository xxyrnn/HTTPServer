# HTTPServer

A simple Java HTTP server

---

## Download

Just clone the repository by running:

```bash
git clone https://github.com/xxyrnn/HTTPServer.git
```

## Usage

1. `cd` into the source folder:

   ```bash
   cd src/server
   ```

2. Start the server by running

   ```bash
   java Server.java
   ```

3. Open your browser and enter `localhost:6789` in the address bar

## Change the default page

To change the default page you have to edit/replace the file called `index.html`.<br />
NOTE: CSS can only be read if inline or through the `<style>` tag in the HTML `<head>`
